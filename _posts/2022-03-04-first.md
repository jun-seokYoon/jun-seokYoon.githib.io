**3. List 와 Slicing**

*3.1) 1차원 리스트의 생성*

    x = [1, 2, 3, 4, 5, 6]

    print(x)

=> [1, 2, 3, 4, 5, 6]

    print(type(x))

=> <class 'list'>

*3.1.1) index *

( = 리스트에서 n번째 숫자 가져오기 list는 0번째부터 시작한다.)

x = [1, 2, 3, 4, 5, 6]

    print(x[2])

=> 3

    print(x[5])

=> 6

*3.1.2) revers index*

( = 리스트의 뒤에서 부터 가져오기 (reverse는 -1 부터 시작한다.)

    print(x[-2])

=> 5

*3.2) 2차원리스트의 생성*

    y = [

        [10, 11, 12, 13, 14, 15],

         [20, 21, 22, 23, 24, 25]

    ]

    print(y)

=> [[10, 11, 12, 13, 14, 15], [20, 21, 22, 23, 24, 25]]

    print(type(y))

=> <class 'list'>

*3.3) 슬라이싱*

( = x[a:b] 는 list x에서 a번째부터 b바로 직전번쨰 까지의 list를 가져오는 것 )

    print(x[0:4])

=> [1, 2, 3, 4]

    print(x[2:])

=> [3, 4, 5, 6]

    print(x[:4])

=> [1, 2, 3, 4]

    print(x[:-3])

=> [1, 2, 3]

    print(x[-3:])

=> [4, 5, 6]
